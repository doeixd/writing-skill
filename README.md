# blog-writing

A writing skill for drafting and editing blog posts that sound like a person thinking on the page instead of generic AI copy.

## What It Does

- drafts blog posts, essays, and commentary with a clear point of view
- removes common AI tells in structure, tone, vocabulary, and rhythm
- keeps prose conversational, spoken, and readable without sounding preachy or self-important
- emphasizes mechanism-first explanation, specificity, and strong editing passes

## Install

Install directly from GitHub with `skills.sh`:

```bash
npx skills add https://github.com/doeixd/writing-skill --skill blog-writing
```

You can also install from the owner/repo form:

```bash
npx skills add doeixd/writing-skill --skill blog-writing
```

## Use

Ask your agent to use the skill explicitly:

```text
Use $blog-writing to draft a blog post in a calm, systems-brained voice that avoids common AI tells.
```
